**Страница**
Параметры:
* глубина хранения данных - как далеко в прошлое можно писать данные
* максимальный размер страницы
* размер кешированных данных в памяти.

Создание хранилища:
1. Создание папки. Если уже есть, то очистка и потом создание.
2. Создание первой страницы.

Открытие
1. Получение списка страниц.
2. Поиск той, что более старая.
3. Подгрузка самой старой.

Запись данных:
1. Блокировка
2. Если страница полная, то создание новой
3. Запись в текущую страницу.

**Построение индексов**
При записи в страницу кешуруются значения (id,time) записываемых значние в виде дерева. 
Потом создается 2 индексных файлы для пар (Id,Pos).idx (Time,Pos).tps
Кеши записывают после закрытия страницы.
При открытии индексы подгружаются вновь.
